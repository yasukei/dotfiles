# usage
bash INSTALL.sh

INSTALL.sh invoke each directory's INSTALL.sh

# directories
## dotfiles
Put dotfiles which are symbolic linked from $HOME.

## apt-get
Put files of apt package name.

## bin
Put scripts (*.sh) to install binaries to $HOME/bin
